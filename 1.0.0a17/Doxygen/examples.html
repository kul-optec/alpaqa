Examples

The following is a list of examples demonstrating how to use the alpaqa library. More Python examples can be found in the Sphinx documentation. An example and tips on how to structure a C++ project that uses alpaqa can be found in examples/CMake/Solver.

C++ examples

  • C++/CustomCppProblem/main.cpp : How to define and solve a custom problem in C++
  • C++/SimpleUnconstrProblem/main.cpp : Similar, but for problems without constraints
  • C++/FortranProblem/main.cpp : How to define a custom problem in Fortran, and how to solve it in C++
  • C++/DLProblem/main.cpp : How to define a problem that can be loaded dynamically, and how to solve it in C++
  • C++/CasADi/Rosenbrock/main.cpp : How to load a pre-compiled CasADi problem, and how to solve it in C++
  • C++/CustomControlCppProblem/main.cpp : How to define and solve a custom optimal control problem (OCP) in C++
  • problems/sparse-logistic-regression.cpp : How to define a problem that can be loaded dynamically, and how to solve it using the alpaqa-driver program
  • C++/Advanced/lasso-fbs.cpp : How to use the alpaqa::prox functions

CMake examples

  • CMake/Solver : How te set up a Conan + CMake project that uses alpaqa

Python examples

  • Python/simple_optimization/getting-started.py : Define and solve a simple nonlinear program in Python
  • Python/simple_optimization/rosenbrock.py : Similar, but also includes a visualization of the iterates
