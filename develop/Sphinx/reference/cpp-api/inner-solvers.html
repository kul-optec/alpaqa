Parameters:

  • problem – [in] Problem description

  • opts – [in] Solve options

  • x – [inout] Decision variable \( x \)

  • y – [inout] Lagrange multipliers \( y \)

  • Σ – [in] Constraint weights \( \Sigma \)

  • err_z – [out] Slack variable error \( g(x) - \Pi_D(g(x) + \Sigma^{-1} y) \)
